   - SVG尺寸: 可缩放矢量图形

   - 文件名: `logo-footer.svg` (主要), `logo-footer.png` (备用)
   - 描述: 深色背景用标志
   - SVG尺寸: 可缩放矢量图形
   - PNG尺寸: 400 x 120px (建议最大尺寸)
   - 主要格式: SVG
   - 备用格式: PNG (带透明度)

> **SVG优势说明**：
> - 无限缩放，在任何设备分辨率下都保持清晰
> - 文件体积更小，提升网站加载速度
> - 可通过CSS调整颜色，适应不同背景
> - 支持动画和交互效果
> - 无需为不同尺寸准备多个版本

## 注意事项

1. 所有图片应采用专业摄影或高质量渲染
2. SVG格式优先用于所有图标、边框和UI元素，PNG作为备用
3. 产品图片应使用透明背景，便于在不同背景下使用
4. 场景图片应体现高端氛围和品牌调性
5. 所有素材应保持一致的色调和视觉风格
6. 人物头像应使用高质量专业照片，不建议使用AI生成图像
7. 所有素材均需版权安全，可商用
